## Git 사전 준비
`$ git config --global user.name '{이름}'`  
`$ git config --global user.email '{이메일}'`

## Git 기초 흐름
```
$ git init //저장소 설정
$ git add .
$ git commit -m '{커밋명}'
$ git branch -M main //default branch 가 main으로 변경되어서..
$ git remote add origin {링크}
$ git push -u origin main
```
